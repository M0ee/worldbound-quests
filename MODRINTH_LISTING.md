## Summary

A shared co-op world-border quest campaign with day-gated progression, escalating challenges, a luxury pixel quest book, and recording-friendly death punishments.

## Description

Worldbound Quests turns Minecraft into a shared survival challenge. Everyone begins inside a tiny world border, works through a 50-quest campaign, unlocks harder objectives over time, expands the playable area, and risks weighted punishments on death.

The mod is made for co-op videos and multiplayer sessions. Quest progress is shared by the server, the quest book shows locked and unlocked objectives, and the HUD tracks the active quest, Minecraft day, current border size, and total expanded area.

## Features

- 50 day-gated campaign quests from first logs to the Ender Dragon.
- Shared server-side quest progress for LAN and dedicated servers.
- Quest book UI opened with `K`, including scrolling, chapter labels, day locks, and progress rows.
- HUD with active quest, Minecraft day, border size, expanded area, and punishments.
- Death punishment wheel designed for multiplayer videos.
- Guide villager hand-in flow for item quests.

## Install

1. Install Fabric Loader 0.19.2 for Minecraft Java 26.1.2.
2. Install Fabric API 0.150.0+26.1.2.
3. Put the Worldbound Quests jar in the server `mods` folder.
4. Put the same jar in every player's client `mods` folder.
5. Start a world or server and run `/worldbound start`.
